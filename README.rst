===================================================================================================
CLI Utilities
===================================================================================================

A collection of my standard CLI scripts and utilities.


Install
===================================================================================================

Scripts are installed to $HOME/bin

Ubuntu
---------------------------------------------------------------------------------------------------

.. code-block:: console

    $ sudo apt update
    $ sudo apt install -y python3-bs4 \
                          python3-dateutil

    $ make install
